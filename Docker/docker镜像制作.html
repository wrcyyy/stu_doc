3.3. Docker镜像相关

使用Dockerfile制作

Dockerfile指南

  1. 编写Dockerfile文件
  2. 在Dockerfile所在目录下执行docker build -t <image_name>:<tag> .

将容器打包为镜像

docker commit [OPTIONS] CONTAINER [REPOSITORY[:TAG]]用于将容器打包为镜像 参数说明：

-a  提交的镜像作者
-c  使用Dockerfile指令来创建镜像
-m  提交时的说明文字
-p  在commit时，将容器暂停

示例：docker commit 201630bd515e registry.com.cn/test_image:v0.0.1
