1.9. python多类继承

类的继承

在日常工作过程中经常需要用到类的继承，单个类的继承还好，遇到复杂一点的就容易出问题

不同父类中存在相同的方法，子类对象在调用时会调用哪一个父类中的方法？

  • 假设有三个类，其中A、B为父类，C为继承了A和B的子类；我们可以调用类的内置属性__mro__方法来查看方法搜索顺序

  • MRO全称method resolution order，主要用于在多类继承时判断方法、属性的调用路径

class A:
    def say_hello(self):
        return '你好！A'


class B:
    def say_hello(self):
        return '你好！B'


class C(A, B):
    def get_name(self):
        return 'wrc'


if __name__ == '__main__':
    print(C.__mro__)

通过执行以上代码可以得到输出：(<class '__main__.C'>, <class '__main__.A'>, <class '__main__.B'>, <class 'object'>)，不难看出：

  1. 子类在调用方法时按照class.__mro__的输出结果从左到右进行顺序查询
  2. 如果在某一个类中找到该方法则直接执行并停止搜索
  3. 如果没有找到该方法则从下一个类中查找
  4. 如果所有类中都没有该方法则报错

多继承的使用

class Human:
    def __init__(self, sex):
        self.sex = sex

    def p(self):
        print('这是Human的方法')


class Person:
    def __init__(self, name):
        self.name = name

    def p(self):
        print('这是Person的方法')

    def person(self):
        print('这是person特有的方法')


class Teacher(Person):
    def __init__(self, name, age):
        super(Teacher, self).__init__(name)
        self.age = age


class Student(Human, Person):
    def __init__(self, name, sex, grade):
        Human.__init__(self, sex)
        Person.__init__(self, name)
        self.grade = grade


class Son(Human, Teacher):
    def __init__(self, sex, name, age, fan):
        Human.__init__(self, sex)
        Teacher.__init__(self, name, age)
        self.fan = fan


if __name__ == '__main__':
    stu = Student('李明', '男', 88)
    print(stu.name, stu.sex, stu.grade)
    stu.p()
    son1 = Son('男', '张三', 19, '赛车')
    son1.person()
    son1.p()

运行上述代码，可以得到如下结果：

李明 男 88
这是Human的方法
这是person特有的方法
这是Human的方法

从这个例子中可以看到：

  • 多类继承时会按照从左到右的顺序查找父类中的方法
  • 多层父类继承时，子类会继承父类的所有方法和属性，包括父类的父类的所有方法和属性

多类继承的注意事项

class Human:
    def __init__(self, sex):
        self.sex = sex

    def p(self):
        print('这是Human的方法')

    def str1(self):
        print(f'这是一名{self.sex}性')


class Person:
    def __init__(self, name):
        self.name = name

    def p(self):
        print('这是Person的方法')

    def person(self):
        print('这是person特有的方法')

    def str2(self):
        print(f'他的名字是{self.name}')


class Student(Human, Person):
    def prin(self):
        print('student')


if __name__ == '__main__':
    stu = Student('男')
    stu.p()
    stu.str1()

子类从多个父类派生且子类没有自己的构造函数时：

  • 按照顺序继承，哪个父类在最前边且有自己的构造函数时，就继承该父类的构造函数
  • 如果最前边的父类也没有自己的构造函数，则向后顺序查找

使用super调用父类属性方法的注意事项

我们先不使用super调用父类的方法，使用父类名.方法名的方式进行：

class Parent:
    def __init__(self, name):
        print('parent的init开始被调用')
        self.name = name
        print('parent的init调用结束')


class Son1(Parent):
    def __init__(self, name, age):
        print('Son1的init开始被调用')
        self.age = age
        Parent.__init__(self, name)
        print('Son1的init调用结束')


class Son2(Parent):
    def __init__(self, name, gender):
        print('Son2的init开始被调用')
        self.gender = gender
        Parent.__init__(self, name)
        print('Son2的init调用结束')


class Grandson(Son1, Son2):
    def __init__(self, name, age, gender):
        print('Grandson的init开始被调用')
        Son1.__init__(self, name, age)
        Son2.__init__(self, name, gender)
        print('Grandson的init调用结束')


if __name__ == '__main__':
    gs = Grandson('张三', 18, '男')
    print(f'姓名：{gs.name}')
    print(f'性别：{gs.gender}')
    print(f'年龄：{gs.age}')

parent的init调用结束

parent的init开始被调用
parent的init调用结束
Son2的init调用结束

现在使用super调用父类中的方法

class Parent:
    def __init__(self, name, *args, **kwargs):
        print('parent的init开始被调用')
        self.name = name
        print('parent的init调用结束')


class Son1(Parent):
    def __init__(self, name, age, *args, **kwargs):
        print('Son1的init开始被调用')
        self.age = age
        super(Son1, self).__init__(name, *args, **kwargs)
        print('Son1的init调用结束')


class Son2(Parent):
    def __init__(self, name, gender, *args, **kwargs):
        print('Son2的init开始被调用')
        self.gender = gender
        super(Son2, self).__init__(name, *args, **kwargs)
        print('Son2的init调用结束')


class Grandson(Son1, Son2):
    def __init__(self, name, age, gender):
        print('Grandson的init开始被调用')
        super(Grandson, self).__init__(name, age, gender)
        print('Grandson的init调用结束')


if __name__ == '__main__':
    gs = Grandson('张三', 18, '男')
    print(f'姓名：{gs.name}')
    print(f'性别：{gs.gender}')
    print(f'年龄：{gs.age}')

执行上述代码后得到以下结果：

Grandson的init开始被调用
Son1的init开始被调用
Son2的init开始被调用
parent的init开始被调用
parent的init调用结束
Son2的init调用结束
Son1的init调用结束
Grandson的init调用结束
姓名：张三
性别：男
年龄：18

结论：

  • 为了避免多类继承报错，我们需要对父类使用不定长参数，接受参数
  • 在单类继承上，super()__init__()和类名.__init__()基本没有差异
  • 在多类继承上，super方法能够保证每个父类的方法只执行一次
